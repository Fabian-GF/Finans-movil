# Funcionalidades

La siguientes es una lista de las funcionalidades disponibles para el usuario de la aplicación:

1. 💳 Administración de cuentas: El usuario puede:
    - Visualizar sus cuentas registradas
    - Consultar el saldo disponible
    - Revisar información básica de cada cuenta

2. 💸 Gestión de transacciones:
    - Registrar ingresos (dinero que entra)
    - Registrar egresos (dinero que sale)
    - Visualizar el historial de movimientos
    - Filtros en el historial
