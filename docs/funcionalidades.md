# Funcionalidades

La siguientes es una lista de las funcionalidades disponibles para el usuario de la aplicación:

1. 💳 Administración de cuentas: El usuario podrá:
    - Abrir una cuenta en el banco
    - Revisar los datos de su cuenta
    - Consultar el saldo de su cuenta
    
2. 💸 Operaciones de intercambio
    - Transferencias de dinero entre cuentas
    - Depósitos
    - Extracciones
    - Registro de operaciones
